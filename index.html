#Coffee

The Groovy CoffeeScript Compiler

Download GitHub Project

Download GitHub Project

#Coffee or Hash-Coffee is an implementation of the CoffeeScript command line written in Groovy. It uses the core JavaScript compiler and Mozilla Rhino as libraries to compile the CoffeeScript source into JavaScript.

Many of the compiler options available in NodeJS are being ported to the Groovy version.

#Coffee can be integrated as a Command Line Tool, JAR dependency and as Maven or Gradle Plugins.
