#Coffee

The Groovy Coffee Compiler

Download GitHub Project

Download GitHub Project

#Coffee or Hash-Coffee is an implementation of the Coffee command line. It is written in Groovy and uses the core JavaScript compiler and Mozilla Rhino as libraries to compile the Coffee style sheet into CSS.

Many of the compiler options available in NodeJS are being ported to the Groovy version.

#Coffee can be integrated as a Command Line Tool, JAR dependency and Maven or Gradle Plugins.
