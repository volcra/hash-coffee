#Coffee

The Groovy CoffeeScript Compiler

#Coffee or Hash-Coffee is an implementation of the Coffee command line. It is written in Groovy and uses the core JavaScript compiler and Mozilla Rhino as libraries to compile the Coffee scripts into JS.

Many of the compiler options available in NodeJS are being ported to the Groovy version.

#Coffee can be integrated as a Command Line Tool, JAR dependency and Maven or Gradle Plugins.

Introduction

#Coffee is developed in Groovy. Why? Groovy is a dynamic language for the JVM. Comes with features that complement the Java development. Syntax is less verbose and Closures give the functional style missing in Java. These and some other features increase developer productivity by reducing boilerplate code when developing console applications.

CoffeeScript is a language that compiles into JavaScript. The code compiles one-to-one into the equivalent JS, and there is no interpreation at runtime.

Here is an example of the equivalent code in CoffeeScript and JavaScript

CoffeeScript 
square = (x) -> x * x
cube   = (x) -> square(x) * x
JavaScript 
var square = function(x) {
  return x * x;
};

var cube = function(x) {
  return square(x) * x;
};

Quick Start

Command-Line

Download and build from source. For instructions go to Download.

Then compile your scripts.

$>build\install\hash-coffee\bin\hash-coffee -c app.coffee

JAR dependency

Under development, not production ready

Maven

<dependency>
  <groupId>org.volcra.coffee</groupId>
  <artifactId>coffee-compiler</artifactId>
  <version>1.0.0-SNAPSHOT</version>
</dependency>

Ivy

<dependency org="org.volcra.coffee" name="coffee-compiler" rev="1.0.0-SNAPSHOT" />

Gradle/Grails

compile 'org.volcra.coffee:coffee-compiler:1.0.0-SNAPSHOT'
